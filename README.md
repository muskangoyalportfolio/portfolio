# Muskan Goyal Portfolio Site

This site is ready for GitHub Pages.

## Edit content
- Open `content.js`
- Update the project titles, page ranges, page captions, or contact details
- Replace any `assets/page-XX.webp` image if you export new pages from the PDF

## Project structure
- `index.html` = home page
- `projects/<project-slug>/index.html` = dedicated project pages
- `assets/` = page images exported from the PDF

## Deploy to GitHub Pages
1. Upload the folder contents to your GitHub repository.
2. In GitHub, enable Pages from the repository settings.
3. Choose the branch/folder that contains `index.html`.

## Notes
- The site uses plain HTML, CSS, and JavaScript for easy editing and GitHub Pages compatibility.
- Each project page opens separately and displays the corresponding page range from the portfolio PDF.
